# AprendiendoFlask
 Ejercicios Basicos para poder interpretar el micro Framework Flask
##Como ejecutar los ejercicios
 PRIMERO que todo existen muchas formas pero esta es la mas sencilla
 1.- Instalar Anaconda para tu sistema operativo
 2.- Ejecutar en tu CMD o Terminal el comando 'conda' para verificar la instalacion
 3.- Ejecutar este comando 'conda create' seguido del nombre del entorno ejemplo: 'conda create entorno'
 4.- Ejecutar el comando con el comando 'conda activated' enseguida el nombre de tu entorno ejemplo 'conda actuvated entorno'
 5.- Navegar desde la linea de comandos hasta el archivo con terminacion .py
 6.- Ejecutar comando python mas el nombre de tu archivo .py
